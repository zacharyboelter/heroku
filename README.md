# the tannin connection

Project 2 for SEI. A place where you can keep track of the wines you love and hate. Record and order by variety, food pairings, origin and more.

Welcome to your very own wine catalog.

# technologies used

*HTML
*CSS
*Bootstrp
*Javascript
*EJS
*Express.js
*MongoDB
*Node.js
*Trello
*Heroku
*InVision
